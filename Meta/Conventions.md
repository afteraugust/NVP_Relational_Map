# Vault Conventions

## File Naming

- **People:** Full name as spoken (e.g., `Rev. James Whitfield`, `Prof. Sarah Lin`)
- **Organizations:** Official name (e.g., `Riverside Community Foundation`)
- **Projects:** Descriptive name (e.g., `Spring Block Party Series`)
- **Funders:** Name of the funding source (e.g., `Riverside Community Foundation Grants`)
- **Interactions:** `YYYY-MM-DD Brief Title` (e.g., `2026-03-19 Coffee with Marcus Bell`)

## Frontmatter Rules

- Always use wikilinks in frontmatter: `"[[Person Name]]"` (quoted to avoid YAML issues)
- Use the exact file name in wikilinks â€” no aliases in frontmatter
- For list fields with no value yet, leave an empty `"[[]]"` placeholder or delete the line
- Every note must have a `type` field and matching tag

## Cadence Values

| Cadence | Meaning | Typical contacts |
|---------|---------|-----------------|
| `weekly` | Your closest active relationships | Key collaborators, active funder contacts |
| `biweekly` | Regular but not constant | Project partners, warm introductions |
| `monthly` | Standard check-in rhythm | Most network contacts |
| `quarterly` | Low-touch, keep alive | Peripheral contacts, dormant connections |
| `as-needed` | No regular rhythm | Team members, one-off contacts |

## Relationship Ownership

- **owner** on a Person note = the team member who owns the primary relationship
- This isn't exclusive â€” both team members may interact with the same contact
- Assign based on who has the warmest relationship or most natural connection
- Review ownership periodically as relationships evolve

## Logging Interactions â€” The Critical Habit

This is the single most important thing you do in the vault. Every interaction you log:
- Updates the relational graph (participants become connected)
- Feeds the cadence tracking system (last-contact is computed automatically)
- Builds institutional memory (searchable history of every conversation)
- Powers relational suggestions (the dashboard finds related contacts to reach out to)

**After every meaningful conversation:** create an Interaction note. It takes 2 minutes and pays compound interest.
