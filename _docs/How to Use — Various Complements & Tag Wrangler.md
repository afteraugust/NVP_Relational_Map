# How to Use Various Complements & Tag Wrangler

These are small utility plugins that reduce friction during daily use. They work quietly in the background.

---

## Various Complements

### What It Does
Autocomplete suggestions as you type. When you're filling in frontmatter or writing in the note body, it suggests:

- **Internal links** — type a few letters and it suggests matching note names, ready to insert as `[[wikilinks]]`
- **Frontmatter values** — when editing a frontmatter field, it suggests values that exist elsewhere in the vault for that field

### How to Use It
Just type. The suggestions appear automatically as a dropdown. Use arrow keys to select, Enter to accept, Escape to dismiss.

### Why It Matters
It prevents typos in note names and frontmatter values. If you have a contact named `MacKenzieSchuller`, you don't have to remember the exact spelling — type "Mac" and it suggests the full name. This keeps your wikilinks consistent, which keeps the graph and ExcaliBrain accurate.

### Settings to Verify
Settings → Various Complements:
- "Internal link complement" → ON
- "Front matter complement" → ON

## Tag Wrangler

### What It Does
Lets you rename, merge, or delete tags across the entire vault from a single right-click.

### How to Use It
1. Open the Tags pane (right sidebar or Cmd+P → "Tags: Show tags")
2. Right-click any tag
3. Options: Rename tag, Merge tag into another, Delete tag

### Why It Matters
Over time, tags can drift. Maybe someone uses `#funder` and someone else uses `#funders`. Tag Wrangler lets you fix this in one action instead of manually editing every file.

### When You'd Use It
- During a periodic vault cleanup (every few months)
- When you notice inconsistent tags in the Tags pane
- When renaming a concept (e.g., changing `#community-development` to `#community-building`)

You probably won't use Tag Wrangler daily. It's insurance for when things drift.
